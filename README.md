# Notes Application

A simple Notes Application built using Flask and SQLite.

## Features
- Add Notes
- Save Notes in Database
- Display Notes on Same Page
- Delete Notes

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Project Structure

Notes Application/
│
├── app.py
├── notes.db
├── requirements.txt
├── README.md
│
├── templates/
│   └── index.html
│
└── static/
    └── style.css

## Run Project

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
